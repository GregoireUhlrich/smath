◆ operator=() [2/3]

Symbol & operator= ( double  t_value)

Tries to associate a number with the Abstract.

If abstract has a type that allows the association of a number then we set the value of abstract using setValue(): this concerns Number, Variable, and CFraction for now. Else abstract is replaced with a pure Number.

Parameters
t_valueNew value for the Symbol.
Returns
A reference to *this
Warning
If we try to set the value of a not directly-valued expr (cos(x)) for example, then the expr is erased and replaced with a Number.

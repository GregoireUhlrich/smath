◆ Replace()

Expr Replace ( const Expr &  expr,
const Expr &  old_abstract,
const Expr &  new_abstract 
)

Replace an Abstract by another in an expr.

This function allows to replace old_abstract by new_abstract in expr. For example Replace(cos(x+x^2),x,y-1) = cos(y-1+(y-1)^2). It leaves the original Abstract invariant and returns another Abstract.

Parameters
exprAbstract in which the replacement takes place.
old_abstractAbstract to search and replace.
new_abstractAbstract by which we replace.
Returns
A new Abstract with the replacement done.

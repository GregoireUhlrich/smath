◆ INF

INF = std::make_shared<Constant>("inf")
static

Represents infinity in the program.

When an infinity appears in the abstract evaluation the program should return INF. INF is treated like a Variable. It is simply here to inform the user that *something bad happened** in the expr.

Bug:
Will not work for many cases, in particular log(0)!=INF. Not yet implemented.
